# vue3-masterclass

## Project setup

```
yarn install
```

### Compiles and hot-reloads for development

```
yarn dev
```

### Compiles and minifies for production

```
yarn build
```

### Lints and fixes files

```
yarn lint
```

### Delete the firebase DB and migrate Fresh Data to Db.

```
yarn mig:db
```
